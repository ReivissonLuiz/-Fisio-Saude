## Pré-requisitos
- Flutter SDK instalado e no PATH
- Google Chrome instalado

## Passos

1. Iniciar o Flutter Web no Chrome
```powershell
cd flutter_app; flutter run -d chrome --web-port=8080
```

O app estará disponível em:
- 🌐 Flutter Web: http://localhost:8080
